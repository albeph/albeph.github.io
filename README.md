# Personal Homepage

A clean and minimalist personal homepage that dynamically fetches and displays GitHub repositories.

## Features

- **Dynamic GitHub Integration**: Automatically loads profile details (avatar, name) and repositories directly from the GitHub API.
- **Latest Commits First**: Repositories are sorted dynamically by their most recent activity (`pushed_at` date).
- **Responsive & System Dark Mode**: Completely responsive layout with automatic styling adjustments to match browser/system preferences.
- **No JS Framework Dependency**: Built with vanilla HTML, CSS, and lightweight JavaScript.
